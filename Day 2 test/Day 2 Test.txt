1. difference between document and window object? 
Ans:
Document Object
* Document Object represents any HTML document or web page that is loaded in the browser. 
* Document Object is loaded inside the window.
* Document Object is the object of window property.
* We can access the document from a window using the window Document.
* Syntax for Document Object is - document.propertyname;


Window Object
* Window Object represents a browser window or frame that displays the contents of the webpage.
* Window Object is the very first object that is loaded in the browser.
* Window Object is the object of the browser.
* We can access the window from the window only.
* Syntax for Window Object is - window.propertyname;
